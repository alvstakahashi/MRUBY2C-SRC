# MRUBY2C-SRC
GISTで公開しているプログラムの環境として公開しているものです。実験的にソースを修正したもので、mrubyとして正しく動作しないものです。

mrubyからCの呼出しのソース一式です。　HEWプロジェクトです。シミュレータで実行できます

もし動かして確認する場合は、HEWのセットアップが必要になります。
HEWのセットアップは以下を参考にしてください。　シミュレータもセットアップされます。
